# Installation

To get started, make sure to setup all the prerequisite tools on your local machine (an installer has not yet been developed).

# Install Rust

For an introduction to Rust, visit the docs website (opens new window).

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
rustup component add rustfmt

# Install Solana

See the solana docs (opens new window) for installation instructions. On macOS and Linux,

sh -c "$(curl -sSfL https://release.solana.com/v1.6.9/install)"

# Install Anchor

Plural is built using Anchor (opens new window). To install the Anchor CLI run,

cargo install --git https://github.com/project-serum/anchor --tag v0.6.0 anchor-cli --locked

# Build and Deploy Plural

First make sure you are in the repo directory,

anchor build

anchor deploy
